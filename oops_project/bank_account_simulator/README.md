# Bank Account Simulator (Java CLI)

A simple command-line banking application built using Core Java.

## Features
- Deposit money
- Withdraw money (checks for insufficient balance)
- Check account balance
- Menu-driven interface using switch-case

## Hardcoded Account
The application starts with a predefined account:

Name: Nishant  
Account Number: 101  
Starting Balance: 5000

These values are currently hardcoded inside `BankApp.java` and can be modified.

## Concepts Used
- Classes and Objects
- Constructors
- Methods
- Switch Case
- Loops
- Scanner for user input

## Project Structure
BankAccount.java → Account logic  
BankApp.java → Main program and menu
