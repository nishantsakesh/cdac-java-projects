# Electricity Billing System (Java CLI)

A simple command-line application that calculates electricity bills based on units consumed.

## Features
- Calculates electricity bill using slab rates
- Takes consumer details as input
- Displays total bill amount
- Simple CLI interaction

## Billing Logic
First 100 units → ₹5/unit  
Next 100 units → ₹7/unit  
Above 200 units → ₹10/unit  

## Example Input
Consumer ID: 101  
Consumer Name: Nishant  
Units Consumed: 250  

## Concepts Used
- Classes and Objects
- Constructors
- If-Else Ladder
- Methods
- Scanner for user input
