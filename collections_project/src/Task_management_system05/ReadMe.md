# Task Manager System (Java)

## Overview

This project is a console-based Java application built to manage daily tasks using a simple CRUD system.

The system allows users to add, view, update, and delete tasks. It was created to practice **ArrayList and object management in Java**.

---

## Concepts Used

* Java Classes and Objects
* ArrayList (dynamic storage)
* CRUD operations
* Menu-driven program
* Basic input handling

---

## Class Structure

### Task

Represents a task.

* taskId
* taskName
* taskStatus

---

### TaskManager

Handles all operations:

* addTask()
* viewTasks()
* updateTask()
* deleteTask()

Stores tasks using:

```text id="8y7k2m"
ArrayList<Task>
```

---

### Main

Provides menu-based interaction.

---

## Features

### 1. Add Task

Create a new task with ID, name, and status.

---

### 2. View Tasks

Display all tasks.

---

### 3. Update Task

Modify task details using task ID.

---

### 4. Delete Task

Remove a task from the system.
